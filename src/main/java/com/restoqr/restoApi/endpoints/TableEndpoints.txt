# Endpoints de Mesas (Table) - RestoQR API

RUTA PRINCIPAL: /tables

GET   LEER TODAS LAS MESAS: /all

GET   BUSCAR UNA MESA POR SU NUMERO: /tables/find/{numero}

POST  CREAR MESAS: /tables/create/number/capacity

PUT   ACTUALIZAR UNA MESA POR SU ID: /api/tables/update/{id}/{number}/{capacity}/{available}

DELETE ELIMINAR UNA MESA: /tables/delete/{id}
